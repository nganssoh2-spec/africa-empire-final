# Africa Empire MVP

**Plateforme familiale intégrée pour la nouvelle génération africaine.**

## 🌍 Vue d'ensemble

Africa Empire est une plateforme web complète conçue pour unir les familles africaines, fournir des opportunités et promouvoir le bien-être spirituel et physique.

## ✨ Modules

- **👨‍👩‍👧‍👦 Famille**: Profils, généalogie et valeurs familiales
- **✨ Spiritualité**: Numérologie, affirmations et méditations
- **💼 Opportunités**: Bourses d'études et annuaire d'emplois
- **🤖 Chat IA**: Assistant personnel avec IA intégrée
- **💪 Santé**: Routines wellness et exercices
- **💰 Finances**: Gestion budgétaire et tracking
- **👥 Communauté**: Forum et espaces de partage

## 🚀 Démarrage rapide

### Installation locale

```bash
git clone https://github.com/nganssoh2-spec/africa-empire-final.git
cd africa-empire-final
npm install
npm run dev
```

Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur.

### Vercel (Production)

[https://africa-empire-final.vercel.app](https://africa-empire-final.vercel.app)

## 📋 Tech Stack

- **Frontend**: Next.js, React, TailwindCSS
- **Deployment**: Vercel
- **Database**: Supabase (ready)
- **API**: Groq AI (ready)
